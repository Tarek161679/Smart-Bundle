# Smart Bundle: MOGA Recommendation Engine 🚀

**Smart Bundle** هو محرك توصيات ذكي مخصص لتطبيقات التجارة الإلكترونية عبر الهواتف المحمولة. يعتمد المشروع على **الخوارزميات الجينية متعددة الأهداف (Multi-Objective Genetic Algorithms)** لتحقيق التوازن الأمثل بين دقة التخصيص وتنوع المنتجات المعروضة.

---

### 🛠 التقنيات المستخدمة (Tech Stack)
*   **Backend:** Node.js & Express.js
*   **Data Processing:** `xlsx` library for Excel parsing[cite: 2]
*   **Algorithm:** Custom Multi-Objective Genetic Algorithm (JavaScript)[cite: 2]
*   **Frontend:** Bootstrap 5 (Mobile-first design)[cite: 2]
